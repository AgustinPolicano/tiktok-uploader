Creator workflow tool

Prepare social media posts from a local dashboard.

Social Publisher Dashboard is a local web dashboard for creators and small teams. It helps prepare post captions, publish approved media to Instagram Business, and upload videos to TikTok inbox as drafts for manual review and posting.

View project repository

Local first

The dashboard runs on the creator's own machine. The app is designed for private content preparation workflows.

TikTok inbox

Videos are uploaded to TikTok inbox as drafts. The creator reviews the video inside TikTok and manually completes the post.

Instagram Business

Approved media can be published to connected Instagram Business accounts using official platform APIs.

Platform use

This app does not directly publish TikTok videos. TikTok content is uploaded as a draft/inbox item using the Content Posting API with the video.upload scope. Final editing and publishing remain under the creator's control inside TikTok.
