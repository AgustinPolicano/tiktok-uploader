Privacy Policy

Last updated: May 19, 2026

1. Overview

Social Publisher Dashboard is designed as a local-first creator tool. The dashboard runs on the user's own machine unless the user chooses to deploy it elsewhere.

2. Data processed by the app

The app may process the following data:

  • Uploaded images and videos selected by the user.
  • Post ideas, captions, hashtags, and editing instructions entered by the user.
  • OAuth tokens and API credentials configured by the user.
  • Basic publishing history, including filenames, timestamps, platforms, and API response identifiers.

3. Local storage

OAuth tokens, upload history, and temporary local files are stored on the machine or server where the user runs the app. The app does not operate a centralized hosted database by default.

4. Third-party services

The app may send data to third-party services configured by the user, including:

  • TikTok APIs, to authenticate the user and upload videos to TikTok inbox as drafts.
  • Meta/Instagram APIs, to publish approved media to Instagram Business accounts.
  • GitHub APIs, to host media files publicly when a public URL is required by Instagram.
  • AI providers, to generate captions when the user configures an AI API key.

5. Public media hosting

When GitHub storage is configured, uploaded media may be committed to a public repository so Instagram can access it through an unauthenticated public URL. Do not upload private, confidential, or sensitive media to public storage.

6. TikTok data

TikTok OAuth is used to obtain an access token with scopes authorized by the user, such as video.upload. The token is used to upload videos to TikTok inbox as drafts. The app does not directly publish TikTok videos.

7. Data deletion

Users can delete local token files, upload history, and temporary files from their own machine. If media was uploaded to a public GitHub repository, users must delete it from that repository to remove public access.

8. Contact

For privacy questions, contact the app owner through the official repository or website where this page is hosted.
